aaa-settings-dialog demo

A modal dialog for soliciting application settings. This demo uses an external json file to specifiy the label, the property of the settings object and the type of control (e.g. slider, toggle) used to solicit the input. A sample settings file is included in the distribution of the element.

Press Open Dialog to display the dialog.
Press Get Settings to see the values set by the dialog.
Press Show Settings Items to see the setting items built from the json file
